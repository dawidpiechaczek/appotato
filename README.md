# appotato

<img width="1024" height="1536" alt="appotato" src="https://github.com/user-attachments/assets/af9c3412-1f33-4749-ad06-03099e24d9be" />

Appotato is a playful mobile app designed to help reduce food waste by tracking product expiration dates. Users can quickly scan barcodes or manually enter items, and the app saves expiration details in an easy-to-manage list. Appotato helps households keep food fresh, save money, and live more sustainably.
